Linux基础指令

ls指令

​ ls是list 的简写，ls指令可以查看文件目录。

  • 可带如下参数：
    1. -l (long 的简写)，表示输出详细信息。这个指令会打印出文件的权限、用户名、文件大小、修改日期、文件名，ls -l可简写为ll；
    2. -a (all 的简写)，表示显示所有文件。包括显示隐藏文件 (以 . 开头)；
    3. -lh (h为human的简写)，-l参数不方便查看，-lh参数是为了方便查看；
    4. --help 查看帮助文档。

cd指令

​ cd是Change Directory的简写，cd指令可以切换文件目录。

​ /表示根目录，即最顶层目录；.表示当前目录；..表示上一级目录。

​ 打开Linux的 terminal（终端）窗口，默认在用户目录，可使用pwd命令查看绝对路径。

​ ~ 一般代表/home/用户目录，Terminal中的 ~ $ 是指输入指令将在 ~ 这个目录下执行。

  • 常用cd 命令：

    1. 去往子文件夹

      示例：~$ cd Documents

      结果：~/Documents$

    2. 返回上一级目录

      示例：~$ cd ..

      结果：~$

    3. 返回之前所在目录

      示例：~$ cd -

      结果：~/Documents$

    4. 返回上级目录的上级目录

      示例：~$ cd ../..

      结果：/home$

    5. 切换到用户目录

      示例：~$ cd ~

      结果：~$

    6. 切换到绝对路径对应目录

      示例：~$ cd /home/zl/Documents

      结果：~/Documents$

mkdir指令

​ mkdir是make directory的简写，mkdir命令可以创建文件夹。

rmdir指令

​ rmdir是remove directory的简写，rmdir命令可以移除文件夹。

​ 注意：要移除的文件夹必须是空文件夹，不然会移除失败。

touch指令

​ touch命令可以创建文件。

  1. 创建一个文件

    ~/Documents$ touch file

  2. 创建多个文件

    ~/Documents$ touch file1 file2 file3

rm指令

​ rm是remove的简写，rm指令可以移除文件或文件夹。

  1. 移除单个文件

    rm file

  2. 有提示地移除文件 (为避免误删)

    可带参数：

    • -i 每个要移除的文件都进行提示
    • -I 移除文件超过3个文件才进行提示

  3. 移除文件夹

    使用参数-r 或 -R (recursively) 用来删文件夹。

    和 rmdir 不同，rm -r 可以在文件夹中有文件的情况下删除这个文件夹。

cp指令

​ **cp是copy的简写，cp命令可以复制文件或文件夹。 **

  1. 复制文件

    cp file file_copy

    注意：如果 file_copy 已经存在，将会直接覆盖已存在的文件。 如果要避免直接覆盖，在cp指令后面加参数-i（interactively）。

  2. 复制文件夹

    cp -R folder/ folder_copy/

    注意：复制文件夹，需要加参数-R （recursively）。

mv指令

​ mv是move的简写，mv指令可以剪切或重命名文件

  1. 剪切文件

    mv file folder/

  2. 重命名文件

    mv file file_rename

nano指令

​ nano是linux的一款文字编辑工具。我们做基本的terminal端文本编辑。

​ nano test.py

cat指令

​ **cat是concatenate的简写，cat指令可以显示文件内容，或者将某个文件内容写入到其它文件。 **

  1. 查看文件内容

    cat test.py

  2. 将文件的内容写入另一个文件

    cat test.py > test1.py

  3. 将多个文件的内容一起写入另一个文件

    cat test.py test1.py > test2.py

  4. 将文件的内容添加到另一个文件的末尾

    cat test2.py >> test3.py

whereis 指令

​ whereis指令可以搜索程序名，并且只搜索二进制文件（参数-b），man说明文件（参数-m）和源代码文件（参数-s）。如果省略参数，则返回所有信息。

which指令

​ which指令可以在PATH变量指定的路径中搜索某个系统命令的位置，并且返回第一个搜索结果。使用which指令可以查看某个系统命令是否存在，以及执行命令的绝对路径。

​ 示例：

​ which python3

​ 结果

​ /usr/bin/python3

​ 如果puthon文件有可执行权限，可以在python文件首行加上#！和python命令的绝对路径，即可在terminal中输入./文件名.py直接执行python文件。

​ 示例：

test.py

文件权限

ls 查看权限

​ 在 Terminal 中输入ls -l或ll命令即可查看文件的权限

​ 文件权限表示如下图所示：

​

  • Type: 最常见的是 - 为文件，d 为文件夹，其它的还有l, n …

  • User: User对应的三个位置表示User身份能对文件做什么处理

    （r 能读； w 能写； x能执行； - 不能完成某个操作）

  • Group: 一个 Group 里可能有一个或多个 user，这些权限的样式和User一样。

  • Others: 除了 User 和 Group 的其它人的操作权限，这些权限的样式和User一样。

chmod 修改权限

​ chmod是change mode的简写，chmod指令可以修改文件权限。

​ 指令格式：

​ $ chmod [谁][怎么修改] [哪个文件]

[谁]

  • u: 对于 User 修改
  • g: 对于 Group 修改
  • o: 对于 Others 修改
  • a: (all) 对于所有人修改

[怎么修改]

  • +, -, = 作用形式：加上，减掉，等于某些权限
  • r, w, x 单个或多个权限一起，比如 r rx

[哪个文件]

  • 施加操作的文件, 可以为多个
