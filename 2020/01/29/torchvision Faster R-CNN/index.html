torchvision Faster R-CNN

项目地址

torchvision Github项目地址： https://github.com/pytorch/vision

创建虚拟环境

  1. 创建python3.6的anaconda虚拟环境（命名为pytorch）
    conda create -n pytorch python=3.6

  2. 激活pytorch虚拟环境
    conda activate pytorch

  3. 安装pytorch、torchvision
    conda install pytorch torchvision torchaudio cudatoolkit=9.2 -c pytorch

前期准备

  1. 安装COCOAPI

  2. 制作数据集
    数据集格式与PASCAL VOC数据集相同，包含JPEGImages和Annotations
    注意：Annotations文件中元素应具有合理性！
    定义数据集类Person.py

  3. 定义模型

  4. 数据增强
    在图像输入到网络前，需要对其进行数据增强。

  • 由于Faster R-CNN模型本身可以处理归一化（默认使用ImageNet的均值和标准差来归一化）及尺度变化的问题，因此无需进行归一化或图像缩放操作。
  • 由于from torchvision import transforms中transforms只能对图片进行数据增强，无法同时改变图片对应的label标签，因此使用torchvision Github项目中封装好的用于模型训练和测试的函数：https://github.com/pytorch/vision/tree/master/references/detection
    需要engine.py、utils.py、transforms.py、coco_utils.py、coco_eval.py五个文件，其中transforms.py里面在对图像进行水平翻转（RandomHorizontalFlip）时变换target（bbox）。
    示例: 编写相应的数据增强函数transform.py

  1. 编写训练文件（train.py）

  2. 编写测试文件（test.py）

训练

CUDA_VISIBLE_DEVICES=0 python train.py

测试

python test.py
