Docker(nvidia-docker)使用教程

Docker国内镜像源

Docker启动、停止、查看运行状态

启动docker

systemctl start docker

关闭docker

systemctl stop docker

查看docker运行状态

systemctl status docker

镜像操作

从仓库中搜索镜像
docker search 镜像名

从仓库获取镜像

docker pull 镜像名:版本号

上传镜像到仓库

docker push 镜像名:版本号

查看本地镜像

docker images

删除本地镜像

docker rmi 镜像名:版本号

容器操作

将镜像运行成一个容器

docker run 镜像名:版本号

参数：
-i 持续运行
-t 分配伪终端
-d 后台运行
-p 外部端口:内部端口 端口映射
-v 外部文件:内部文件 文件映射
–gpus 指定gpu
–shm-size 共享内存大小
–name 运行容器名称
示例：docker run -idt -p 30000:22 -p 6006:6006 -p 8097:8097 -v /home:/home –gpus all –shm-size 64G –name ubuntu ubuntu:16.04

查看正在运行的容器

docker ps

查看所有容器

docker ps -a

启动容器

docker start 容器名或ID

停止容器

docker stop 容器名或ID

重启容器

docker restart 容器名或ID

删除容器

docker rm -f 容器名或ID

进入容器

docker exec -it 容器名或ID bash

退出容器

exit

将容器保存为镜像

docker commit 容器名或ID 镜像名

镜像保存和加载

将镜像保存为tar文件

docker save 镜像名 > 保存文件名.tar

从tar文件中加载镜像

docker load < 文件名.tar

通过Dockerfile构建镜像

Docker build -t 镜像名:版本号 Dockerfile

宿主机与容器之间拷贝文件

宿主机拷贝文件到容器

docker cp 宿主机文件 容器名或ID:容器拷贝路径

容器拷贝文件到宿主机

docker cp 容器名或ID:容器文件 宿主机拷贝路径

ssh连接docker容器

  1. 映射22号到宿主机端口
    -p 宿主机端口:22

  2. 修改容器内root用户密码
    passwd root

  3. 安装openssh-server
    apt-get install openssh-server

  4. 修改配置文件
    vim /etc/ssh/sshd_config

  5. 重启ssh服务
    ssh restart
