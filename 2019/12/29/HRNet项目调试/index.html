HRNet项目调试

项目地址

https://github.com/leoxiaobin/deep-high-resolution-net.pytorch

根据README.md创建虚拟环境

  1. 克隆项目并进入项目文件夹

  2. 创建python3.6的anaconda虚拟环境（命名为HRNet）
    conda create -n HRNet python=3.6

  3. 激活simple-HRNet虚拟环境
    conda activate HRNet

  4. 安装实验需要的软件包
    pip install -i https://mirrors.bfsu.edu.cn/pypi/web/simple -r requirements.txt

前期准备

  1. 编译库文件

  2. 安装COCOAPI

  3. 创建output文件夹和log文件夹

    文件夹结构如下

  4. 下载预训练权重至/models/pytorch/pose_coco文件夹下
    预训练权重：pose_hrnet_w48_384x288.pth

  5. 准备HRNet数据集
    将制作好的数据集放至./data/coco文件夹下，包含annotations和images两个文件夹，images文件夹下包含train2017和val2017两个子文件夹，数据集格式与MSCOCO相同

训练

  1. 编辑配置文件
    vim experiments/coco/hrnet/w48_384x288_adam_lr1e-3.yaml
    更改GPUS、PRETRAINED、BATCH_SIZE_PER_GPU

  2. 取消tensorboardX使用
    注释掉tools/train.py中的111行代码
    # writer_dict['writer'].add_graph(model, (dump_input, ))

  3. 开始训练
    CUDA_VISIBLE_DEVICES=0 python tools/train.py --cfg experiments/coco/hrnet/w48_384x288_adam_lr1e-3.yaml
