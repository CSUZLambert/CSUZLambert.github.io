YOLOv3项目调试

项目地址

https://github.com/eriklindernoren/PyTorch-YOLOv3

根据README.md创建虚拟环境

  1. 克隆项目并进入项目文件夹

  2. 创建python3.6的anaconda虚拟环境（命名为YOLOv3）
    conda create -n YOLOv3 python=3.6

  3. 激活YOLOv3虚拟环境
    conda activate YOLOv3

  4. 安装实验需要的软件包
    pip install -i https://mirrors.bfsu.edu.cn/pypi/web/simple -r requirements.txt

前期准备

  1. 下载预训练权重

  2. 创建自定义模型（生成yolov3-custom.cfg）

  3. 设置类别名称
    vim data/custom/classes.names每个类别占一行

  4. 准备个人数据集

  • （1）图像放在data/custom/images文件夹中
  • （2）注释文件放在data/custom/labels文件夹中
    注：注释文件中的每一行都对应一个边界框，具有label_idx x_center y_center width height五个参数，其中坐标应该被缩放到[0, 1]（即：坐标值/对应宽高）
  1. 定义训练集和验证集
    生成对应的data/custom/train.txt文件和data/custom/valid.txt文件

训练

python3 train.py --model_def config/yolov3-custom.cfg --data_config config/custom.data --pretrained_weights weights/darknet53.conv.74

测试

CUDA_VISIBLE_DEVICES=0 python detect.py --model_def config/yolov3-custom.cfg --weights_path checkpoints/yolov3_ckpt_99.pth --class_path data/custom/classes.names
