CUDA、cudnn安装

下载并安装CUDA和补丁包

  1. 下载地址：https://developer.nvidia.com/cuda-90-download-archive?target_os=Linux&target_arch=x86_64&target_distro=Ubuntu&target_version=1604&target_type=runfilelocal

  2. sh执行安装（注意：不安装显卡驱动）
    sudo sh cuda_xxx_linux.run

  3. 修改环境变量（修改～/.bashrc文件）

    执行source ~/.bashrc生效

  4. 检查当前的cuda版本
    nvcc -V

下载并安装cudnn

  1. 下载地址：https://developer.nvidia.com/rdp/cudnn-archive

  2. 解压cudnn压缩文件
    tar -xzvf cudnn-9.0-linux-x64-v7.tgz

  3. 安装（拷贝头文件和库文件并给予权限）

  4. 查看当前cudnn的版本
    cat /usr/local/cuda/include/cudnn.h | grep CUDNN_MAJOR -A 2

测试

  1. 切换到cuda/samples文件夹
    cd /usr/local/cuda/samples/1_Utilities/deviceQuery/

  2. 编译
    make

  3. 执行
    ./deviceQuery

  4. 验证
    Result = PASS 即成功
